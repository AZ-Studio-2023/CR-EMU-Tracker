API文档

简介

本站提供对外的API接口服务，可以查询到和前端一样的担当数据。目前暂时没有作任何限制措施，但未来可能增添限速等功能。

本站服务器有防御，请不要过度爬取，以免影响正常服务、扣光我们的经费或造成自己IP被封。需要批量数据以供开展研究或其余用途的，可以联系作者或自行部署。

接口

GET/POST https://crtracker.azteam.cn/api/query

请求参数可以以多种方式传递，如Query Args，JSON，application/form等。以Query Args优先。

请求示例

{
    "keyword": "CR400BF-5033"
}
